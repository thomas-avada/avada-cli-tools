# Usage
<!-- usage -->
```sh-session
$ npm install -g avada-cli
$ avada-cli COMMAND
running command...
$ avada-cli (-v|--version|version)
avada-cli/0.0.0 win32-x64 node-v10.15.3
$ avada-cli --help [COMMAND]
USAGE
  $ avada-cli COMMAND
...
```
